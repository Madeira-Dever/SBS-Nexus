# Sistema Bancário Seguro Nexus
SBS Nexus é um projeto construido usando as técnologias mais simples do front-end.

| HTML | CSS | JS | NODE | REACT |
| :---: | :---: | :---: | :---: | :---: |
| :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: | :heavy_multiplication_x: | :heavy_multiplication_x: |

Criado para ser um simulador de banco, possuindo as funcionalidades mais básicas, tais como:

- **criar uma conta**
- **gerar um saldo e número de conta totalmente aleatório**
- **depositar um valor**
- **sacar um valor**
- **transferir um valor**
- **ver as movimentações feitas**
- **além de ter um relógio imbutido**
